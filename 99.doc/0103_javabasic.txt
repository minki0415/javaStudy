[학습노트]

1. 프로그램 개발 사유
	- 사용자(end user, client) 유입
	- 사용자들에게 최적화된 서비스
	
	- 개발 및 서비스를 위한 고려 사항
		1) 요구사항 파악
		2) 서비스 구체화 되면 설계
			- 장비 & 개발 인력
		3) 개발 완료 후 서비스
		...
	
	- 프로그램에 있어서 가장 중요한 것?
		1) 데이터
			- 변수 
				- 다양한 타입
					문자, 문자열, 실수, 정수, 논리 ..
		2) 데이터어떻게 처리할 지에 대한 기능(서비스)
			- 메소드(함수)
				- 예시
					계산기 기능(사칙연산만 가능한 단순 계산기)
						- 숫자값 저장 가능한 변수(메모리)
						- 사칙 연산용 기능의 메소드(4개)
						사칙연산
						수행 : 연산을 위한 값들(실수?, 정수?)
						
2. 필수 암기 사항 
	1) 소스 구조
		- 확장자 : *.java
		- java 소스의 구성
			 package 선언구
			 	- 현 소스의 저장 경로
			 	- package명 : 디렉토리 구조 및 이름
			 	
			 import 선언구
			 	- 다른 package의 프로그램을 가져와서 사용하겠다는 의미
			 	
			 class 이름{
			 	변수(데이터)
			 	생성자(객체 생성 시 호출되는 필수 구성 요소)
			 	메소드(기능)
			 }
			 
			 
		- main()
			브라우저 없이 실행되는 일반 자바 프로그램의 시작 point 기능
			특별 기능의 정해진 메소드(스펙)
			public static void main(String[] args)
			 

3. 문법
	1) 변수
		타입 변수명;
		타입 변수명 = 값;
		
	2) 생성자
		클래스명([arguments]){
			[logic]
		}
		
	3) 메소드
		반환타입 메소드명([arguments]){
			[logic] 
			[return 값;]
		}
